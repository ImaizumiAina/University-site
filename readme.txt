ターゲット：
「優秀な学生＝やる気のある、INIADでの勉強意欲のある学生」ととらえ
勉強意欲があり、INIADでできること、できないことを把握した上で
東洋大学情報連携学部に入学したいと思う日本人学生。

コンセプト：
INIADの受験生の多くがINIAD第一志望になってほしい。
綺麗事で済ませないサイト。


特徴：
・サイトのトップページにすぐ飛ぶことができない
・遊びたいだけの学生は大学に間違えて入り後悔しないような配慮
・できる学生には、問題を与えサイトに訪れるだけで考えることができる
・画像や動画が少ないので読み込みが早い
→現在のINIADのサイトが動画が再生される仕組みになっており、とても読み込みに時間がかかる上に重たかったから。
　また、INIADにとても興味がある学生は文字が多くても読むので文字を多くした。
　一方でそこまで興味は無かったけど、というひとは読まずにサイトを離れてしまう可能性もある。
・大学院の情報や外部向けの情報は排除し、大学だけの情報に絞った。
　本当は学校生活なども入れたかったが、そちらがメインになってしまう心配があったため今回はなしにした。
・ヘッダーをシンプルなものに絞り、INIADの公式サイトにあるようなページ構造が複雑なものにならないよう配慮した。
